May Day Quiz

Campaign Contribution Web Page

  1. Create a lab19 directory with a contrib.html file in it in your LABS directory.
  2. Create a form in your contrib.html page with 3 input elements:
    1. A text input with id attribute name (labeled appropriately).
    2. A text input with id attribute amount (labeled appropriately).
    3. A button input with id attribute button and value attribute make payment.
  3. Write JavaScript to accomplish the following when the button is pushed:
    1. Bundle the values of the name and amount fields in a Contribution object.
    2. Reset the values of both fields to the empty string.
    3. Add the amount to a running total. Display this total on you page.
    4. Add the Contribution (some string containing the name and the amount) to a list of Contributions displayed on your page.
    You may assume that the name and amount fields have appropriate values when the button is pushed.
  4. (Extra Credit) Include the date of each Contribution in the list of Contributions. (Hint: google JavaScript Date. www.w3schools.com seems to be a good source of information.)
  5. (Extra Credit) Use CSS to display your page appropriately.
  6. Edit the index.html file in your LABS directory to have a link to your contrib.html page and synchronize with your web site.
  7. Show Bowen your contrib.html page on your web site before leaving class today.

Always turn off your computer and screen before leaving the room.
