Quiz 2

Given the following HTML and CSS

		<html>
		<head>
			<style type="text/css">
				body{
					color: #00ff00;
				}
				.class1{
					color: #ff0000;
				}
				.class2{
					color: #0000ff;
				}
			</style>
		</head>
		<body>
			<p class="class1">Hello World</p>
			<p class="class2">Hello World</p>
			<p>Hello World</p>
		</body>
		</html>
  1. What is the color of 'Hello World' in the first p tag?
  2. What is the color of 'Hello World' in the second p tag?
  3. What is the color of 'Hello World' in the third p tag?
