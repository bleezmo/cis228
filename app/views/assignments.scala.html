Assignment 5

This program will allow you to click on a map and get the distance between that point and the ietf headquarters

  • First follow the google maps api instructions to initialize a map. Sample code can be seen here. Below is a screenshot

  • When the map is clicked on you will be able to get the latitude/longitude of the location clicked via the 'mouseup' event listener. For example:

    google.maps.event.addListener(map, 'mouseup', 
				function(pt){
					//add code here
				});

    Where 'map' is the maps object you created in the first part, 'mouseup' is the event, and 'function(pt)...' is the event handler passed in.

    You can find the documentation for event listeners here.

  • Within the 'mouseup' event handler (the 'add code here' part), you will manipulate the dom so that two pieces of information show up to the right of the map.

    1. The latitude and longitude of the mouse point clicked
    2. The distance from that location clicked to the IETF headquarters

    You can get the distance from the IETF headquarters in two different ways.

    1. (Not preferred way)Find the location of the IETF headquarters and use the haversine distance formula to calculate the distance between two points

    2. (preferred way)Use an AJAX request to http://cis228.herokuapp.com/api/ietf/distance to get the distance as the response.

      For example, If you wanted to calculate the distance between the ietf headquarters and the location (in lat/long) (40.979027939585436, -73.89541625976562) the request url would be 'http://cis228.herokuapp.com/api/ietf/distance?lat=40.979027939585436&lon=-73.89541625976562' (you can try it out in your browser)
