1. From the start button, open up FileZilla.
  2. In the Local site: window, enter the absolute path to the folder containing your CIS228 folder.
  3. In the Host: window, enter ec2-54-85-69-173.compute-1.amazonaws.com.
  4. In the Username: window, enter the first letter of your first name, followed by your last name.
  5. In the Password: window, enter the last four digits of your emplid.
  6. Click the Quickconnect button. Any files you place in your root directory will be displayed in http://ec2-54-85-69-173.compute-1.amazonaws.com/spring2014/[your first name]

For example, If I upload index.html into my root directory (josh), I will be able to view the file on the web at http://ec2-54-85-69-173.compute-1.amazonaws.com/spring2014/josh/index.html
