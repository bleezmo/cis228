1. From the start button, open up FileZilla.
  2. In the Local site: window, enter the absolute path to the folder containing your CIS228 folder.
  3. In the Host: window, enter ec2-54-204-159-148.compute-1.amazonaws.com.
  4. In the Username: window, enter your first name (all lower case).
  5. In the Password: window, enter your given password.
  6. Click the Quickconnect button. Any files you place in your root directory will be displayed in http://ec2-54-204-159-148.compute-1.amazonaws.com/fall2013/[your first name]

For example, If I upload index.html into my root directory (josh), I will be able to view the file on the web at http://ec2-54-204-159-148.compute-1.amazonaws.com/fall2013/josh/index.html
